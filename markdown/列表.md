# 列表

## 1. 无序列表使用【*+空格】
```
* 无序列表
* 无序列表
* 无序列表
```

* 无序列表
* 无序列表
* 无序列表


## 2. 有序列表使用【数字+.+空格】
```
1. 有序列表1
2. 有序列表2
3. 有序列表3
```

1. 有序列表1
2. 有序列表2
3. 有序列表3


## 3. 列表嵌套：子列表中的选项（选项用【+ | -】）前面添加四个空格
```
1. 有序列表1
    + 这是一个子列表
    + 这是一个子列表
2. 有序列表2
    - 这是一个子列表
    - 这是一个子列表
```

1. 有序列表1
    + 这是一个子列表
    + 这是一个子列表
2. 有序列表2
    - 这是一个子列表
    - 这是一个子列表
